如何在 Mac 上的 Safari、Opera、Chrome、Firefox 和 Edge 浏览器中强制硬刷新

linchun 
2022-05-15 (Updated: 2023-03-03)
技术 › 经验技巧
Mac, 浏览器

在 macOS 系统调试网页，发现各浏览器的强制刷新快捷键是有区别的，总结如下：

1、Safari、Opera

Command (⌘) + Option (⌥) + R

2、Google Chrome、Mozilla Firefox、Microsoft Edge

Command (⌘) + Shift(⇧) + R
